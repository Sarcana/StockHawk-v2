About Us

StockHawk is a real-time stock tracker app that allows users to track the latest data on Nifty 50 stocks using Django and Celery. The app uses the Yahoo Finance API to retrieve real-time stock prices, as well as other important metrics such as change and percent change. Users can select from a range of Nifty 50 stocks and view their real-time data in an easy-to-use interface. The app is designed to provide a convenient and efficient way for users to stay informed about the latest market trends and make informed investment decisions

Why StockHawk?

The stock market is a complex and dynamic environment, with stock prices fluctuating constantly in response to market conditions and other factors. To succeed in this environment, it's essential to have access to real-time data on stock prices and other metrics. The real-time stock tracker app provides users with exactly that, offering a range of features designed to make it easy to track the latest data on Nifty 50 stocks..
